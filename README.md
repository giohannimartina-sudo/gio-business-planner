# GIO Business Planner v20 LIVE PRO

Deze versie gebruikt jouw juiste v20 HTML als basis en draait binnen een Next.js/Vercel project.

## Inhoud
- Dashboard
- Planning
- Klanten met wijzigknop
- Projecten
- Uren
- Materiaal
- Uitgaven / Investeren
- Vrijedagen
- Balans
- Export
- JSON back-up / import

## Belangrijk
Gegevens worden in de browser opgeslagen via localStorage. Maak dagelijks een JSON-back-up via Export.
Supabase-cloudopslag kan als volgende stap worden aangesloten zonder je v20 workflow te verliezen.
