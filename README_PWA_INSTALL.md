# GIO Business Planner - PWA Installer update

Deze versie bevat:
- jouw G-logo als app-icoon
- manifest.webmanifest
- service worker sw.js
- Android install ondersteuning
- shortcuts: Inklokken, Projectkaart, Backup

Na upload naar GitHub en Vercel:
1. Open de live link in Chrome op Android.
2. Sluit alle oude tabs van de planner.
3. Open opnieuw.
4. Tik op ⋮ rechtsboven.
5. Kies Install app of Toevoegen aan startscherm.
6. Als alleen Toevoegen zichtbaar is: kies die. De app opent daarna alsnog fullscreen met jouw logo zodra Chrome de PWA-cache herkent.

Belangrijk:
Verwijder eerst oude snelkoppelingen van je startscherm voordat je opnieuw toevoegt.
