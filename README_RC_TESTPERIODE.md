# GIO Business Planner PRO v2.1 RC – Testperiode

Toegevoegd zonder bestaande functies te verwijderen:
- Start- en einddatum per project
- Automatische meerdaagse planning
- Eigen kleur per project
- Rode uitloopstatus met aantal dagen
- Analyse-pagina met geplande/werkelijke dagen, uitloop, uren, omzet, kosten en openstaand
- Compacte actie-iconen naast elkaar met hover-overlay
- Altijd bevestiging bij verwijderen via de centrale verwijderfunctie
- Bestaande Cloud, Materiaal, Uitgaven, KM, Offerte en Facturatie blijven behouden

Gebruik deze versie gedurende de testperiode en maak regelmatig een JSON-back-up.
