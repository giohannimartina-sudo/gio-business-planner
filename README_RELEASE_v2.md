# GIO Business Planner PRO v2.0

Gebouwd op de MASTER-versie. Bestaande functies blijven behouden.

Toegevoegd:
- Dashboard agenda overzicht
- Offerte PRO
- Facturatie PRO met correcte berekening
- Project Archief
- Instellingen met licht/donker thema
- Duidelijke navigatieknoppen
- Inklokken blijft werken via bestaande activeClock opslag

Gebruik:
1. Maak eerst JSON backup in de live planner.
2. Upload deze volledige projectmap naar GitHub.
3. Commit: GIO Business Planner PRO v2.0
4. Laat Vercel deployen.
5. Test klanten, uren, materiaal, uitgaven en cloud.
