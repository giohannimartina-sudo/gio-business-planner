# GIO Business Planner PRO v2.0 – stabiele werkversie

Gebouwd op de gekozen master.

Belangrijkste punten:
- bestaande data en modules behouden
- inklokken blijft actief tot uitklokken
- Cloud login met sessieverversing en wachtwoord tonen
- factuur volgens Excel-template, met eigen logo
- keuze wat klant ziet: uren, tijden, materiaal, leveranciers, bonnen, notities, betalingen, uitgaven
- wijzigen/notitie/verwijderen voor werkregels
- uitgaven bereikbaar op mobiel
- light/dark modus hersteld
