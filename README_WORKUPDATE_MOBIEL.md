# GIO Business Planner PRO - Mobiele werkupdate

Deze update is gebaseerd op de stabiele v2.1 RC KM Slim Update.

## Opgelost
- De foutieve script-tag waardoor JavaScript als tekst op het scherm verscheen.
- De cloud/KM-fout `gioKmFillSavedLocationSelects is not defined` doordat de KM-code nu weer correct wordt geladen.
- Archiefcijfers zijn leesbaar op witte kaarten.
- Betaalde projectregels gebruiken groen `#3D7D30`.

## Mobiel
- Compacte app-layout zonder mega-knoppen.
- Alles blijft binnen de schermbreedte.
- Geen pagina-brede horizontale overflow.
- Normale knophoogte en compacte vaste navigatie.
- Kaarten hebben een subtiel overlay-/zweefeffect.

## Niet gewijzigd
- Desktopindeling.
- Supabase cloudfuncties en gegevensstructuur.
- Bestaande klanten, projecten, uren en materialen.
